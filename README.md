# Sistema de Gestión de Inventario (Java)

Un proyecto de consola desarrollado en Java para simular las operaciones básicas de un inventario. Este repositorio demuestra la aplicación práctica de los pilares de la Programación Orientada a Objetos (POO).

## Tecnologías y Conceptos Aplicados
* **Lenguaje:** Java.
* **Paradigma:** Programación Orientada a Objetos (Clases, Objetos, Encapsulamiento con Getters/Setters).
* **Estructuras de Datos:** Manejo de Colecciones (`ArrayList`).

## Cómo ejecutar el proyecto
1. Clonar este repositorio en tu máquina local.
2. Abrir el proyecto en tu IDE de preferencia (como NetBeans, IntelliJ o Eclipse).
3. Ejecutar el archivo `Main.java` para iniciar la simulación en la consola.
